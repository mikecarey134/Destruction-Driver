* Destruction Driver is a mobile side scrolling game

* Version 0.1.0

### How do I get set up? ###

* Install Unity 3D 5.x.x and up
* Download this repo
* Open it in the Unity editor

* Deployment instructions
   Android
   -Build .apk file in the Unity Editor
   Linux/PC
   -Build executable in the Unity Editor
